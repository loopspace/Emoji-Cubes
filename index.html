Instructions

This webpage makes a reusable set of "story dice" from a given set of characters (including emoji). Using the form below, select a pre-existing collection of symbols (can be none) and add additional characters, then choose the number of dice. Then click on the generate URL button to get a re-usable URL for your dice.

After clicking on the link you can click on the 🖉 icon to get back to this page with the form filled in with that set of dice for editing. If you picked a collection it will be converted to a list of symbols so that you can easily edit it.
