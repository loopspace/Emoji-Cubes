Instructions

This webpage makes a reusable set of "story dice". You configure the dice using the form below. There are some pre-defined collections of symbols to choose from (can be none) to which you can add additional symbols or longer words or phrases. The symbols in the first input box are split into individual characters (hopefully it gets multi-part emoji correct!), whereas in the second box the text is split by commas. In the second box, any text between dollar signs is interpreted as an equation, so $x^2 + y^2 = z^2 will appear as $x^2 + y^2 + z^2$. Then choose the number of dice and click on the generate URL button to get a re-usable URL for your dice.

After clicking on the link you can click on the 🖉 icon to get back to this page with the form filled in with that set of dice for editing.
